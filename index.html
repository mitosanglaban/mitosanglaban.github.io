Kilalanin ang Mitosanglaban

Ang Mitosanglaban ay isang fighting game at open toolkit na naglalahad ng mga kwentong-bayan at mitolohiya ng Pilipinas. Layunin namin na pagyamanin ang lokal na sining, kwento, at game development culture sa bansa — mula sa malikhaing nota ng mga tradisyunal na artista hanggang sa mga nagsisimulang developer na gustong matuto at mag-mod.

Ang site na ito ay showcase ng lore, mga larawan ng kabanata, at mga tool na puwede mong i-download at gamitin.

Bakit kakaiba kami?

  • Iba’t ibang mundo at kultura: Nakatuon sa puso ng Filipino mythos—mas malapit at mas homegrown kaysa sa ibang laro.
  • Estilong sining: Ang key art para kay Alyas at mga halimaw ay mano-mano ginawa at na-scan—pinapahalagahan ang tradisyunal na paggawa.
  • Modular at bukas: Hindi closed-source—puwede i-mod, palawakin, at gawing sarili ng sinuman. May kasamang beginner toolkit para sa mga naghahangad gumawa ng sariling laro.

Mga Proseso: Pag-install, Paglalaro, at Gumawa ng Sariling Laro

A. Proseso para makapag-install at makapaglaro ng Mitosanglaban (PC)

  1. I-download ang zip file: i-click ang Download Game button sa itaas. Ang file ay dapat pinangalanang mitosanglaban-game.zip at naka-host sa https://mitosangalaban.github.io/.
  2. Kapag natapos ang pag-download, i-extract ang zip sa isang folder (kanilang right-click → "Extract all" o gumamit ng 7-Zip/WinRAR).
  3. Buksan ang folder at hanapin ang README.txt o run.bat/mitosanglaban.exe (depende sa build). Sundin ang mga tagubilin sa README.
  4. Kung kinakailangan, i-install ang anumang runtime na nakalista (hal. Microsoft Visual C++ Redistributable). Sundin ang mga link sa README kung may kulang na dependencies.
  5. Patakbuhin ang laro sa pamamagitan ng pag-double click sa executable (Windows: mitosanglaban.exe) at i-configure ang controls sa settings ng laro kung kinakailangan.
  6. Kung may problema, buksan ang logs folder at i-padala ang nilalaman sa amin para sa suporta.

Tandaan: Kung ginamit mo ang GitHub Pages, siguraduhing inilagay ang zip file sa root o sa /assets/ at i-enable ang Pages para sa direktang pag-access sa https://mitosangalaban.github.io/mitosanglaban-game.zip.

B. Proseso para sa mga bata: Gumawa ng sarili mong maliit na laro gamit ang Fighting Game Developer Toolkit

  1. I-download ang Fighting Game Developer Toolkit sa pamamagitan ng Download Fighting Game Developer Toolkit button. Tiyaking ang file ay mitosanglaban-toolkit.zip.
  2. I-extract ang toolkit zip sa isang folder. Sa loob, makikita mo ang mga simpleng halimbawa (sample projects), mga larawan, at isang README.md na may step-by-step na gabay.
  3. Buksan ang folder ng isang sample project. Karaniwan may scenes, sprites, at config files—subukan palitan ang mga larawan sa sprites para makita ang pagbabago sa laro.
  4. Sundin ang halimbawang how-to-play.txt o README.md para patakbuhin ang sample (madalas may run.bat o index.html para sa browser build).
  5. Simulan sa maliit: palitan ang pangalan at hitsura ng isang karakter, baguhin ang bilis ng paggalaw, at subukan kung paano lumalaban sa simpleng AI.
  6. Kapag komportable ka na, pagsamahin ang bagong mga sprites at sundin ang dokumentasyon para gumawa ng sariling scene at talunin ang unang boss.
  7. I-save ang proyekto at i-compress (zip) kung gusto mong ibahagi o i-upload sa GitHub para ipakita sa iba.

Gabay para sa magulang/guro: I-monitor ang paggamit at tulungan ang bata sa unang mga hakbang—ang toolkit ay ginawa na simple at modular para sa madaling pag-aaral.
